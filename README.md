A modern, feature-rich issue tracking system built with vanilla JavaScript, HTML5, and CSS3. Perfect for teams to track bugs, manage projects, and collaborate effectively.


## Features

### Authentication
- Secure login system with demo credentials
- Session management using localStorage
- Protected routes and authentication checks

### Dashboard
- **Real-time Statistics**: Total issues, open, resolved, and overdue counts
- **Kanban Board**: Visual issue tracking with drag-and-drop like columns
- **Quick Actions**: Create new issues directly from dashboard
- **Issue Overview**: Quick view of all issues by status

### 🎯 Issue Management
- **Create Issues**: Comprehensive form with all necessary fields
- **Edit Issues**: Full CRUD operations with status updates
- **Delete Issues**: With confirmation dialog
- **Issue Details**: Modal view with complete information
- **Comments System**: Add and view comments on issues
- **Priority Levels**: Low, Medium, High with color coding
- **Status Tracking**: Open, In Progress, Resolved, Overdue

### Team Management
- **People Directory**: Manage team members
- **Role Assignment**: Assign roles to team members
- **Workload Overview**: See reported vs assigned issues per person
- **Contact Information**: Email and role display

### Project Management
- **Project Directory**: Create and manage projects
- **Project Statistics**: Issue counts per project
- **Custom Keys**: Project identifiers for easy reference
- **Description Field**: Add project details

### Advanced Features
- **Search & Filter**: 
  - Search by issue summary/description
  - Filter by status, priority, project, reporter, assignee
  - Date range filtering
  - Active filter tags with easy removal
- **Sorting**: Sort by any column (ID, Summary, Priority, etc.)
- **View Options**: 
  - Table view with all details
  - Group view by reporter
- **Dark/Light Mode**: Toggle between themes with persistence
- **Responsive Design**: Works on desktop, tablet, and mobile
